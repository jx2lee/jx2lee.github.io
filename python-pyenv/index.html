[Python] Python 환경 구축 with pyenv

Posted on 2021-01-10 Edited on 2021-01-10 In Python Disqus:

MacOS 에서 pyenv 를 이용해 Python 개발 환경을 구축한다.

Install brew

$ ruby -e "$(curl -fsSL [https://raw.githubusercontent.com/Homebrew/install/master/install](https://raw.githubusercontent.com/Homebrew/install/master/install))"

이미 존재하면 homebrew 설치는 생략한다.

Install pyenv

$ brew install pyenv

Set env

export PATH

$ echo 'eval "$(pyenv init -)"' >> ~/.zshrc

  • PATH 환경 변수를 관리하기 위해 pyenv를 시작하고 환경에 나타나는 첫 번째 버전과 달리 실행할 Python 버전을 삽입할 수 있다고 한다.

    • 환경변수 세팅을 안하고 진행을 해보니, 나중에 virtualenvwrapper 패키지 사용 시 virtual path not found error 를 반환하므로 꼭 추가하시길 바랍니다.

  • bash 환경일 경우 .bashrc 에 삽입하도록 한다.

추가 설치 패키지 (권장)

$ brew install zlib sqlite

  • zlib 압축 알고리즘과 SQLite 데이터베이스는 pyenv의 종속된 패키지

  • 올바르게 구성되지 않을 경우 빌드 문제를 일으키는 경우가 많다고 한다. 설치 후 아래와 같은 환경 변수를 적용한다.

Install Python

$ pyenv install —list 명령어로 설치 가능한 리스트를 확인하고 설치를 진행한다.

global python 설정 후 가상환경 세팅 (with virtualenvwrapper)

$ pyenv global 3.7.9

  • python 버젼을 3.7.9 로 세팅

$ $(pyenv which python3) -m pip install virtualenvwrapper

  • global python 버젼으로 virtualenvwrapper 패키지 설치

Set config

이후 mkvirtualenv 커맨드를 통해 가상환경을 생성하고 작업을 진행하면 된다.
