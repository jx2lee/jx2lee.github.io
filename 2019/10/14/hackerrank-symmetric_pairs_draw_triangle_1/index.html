Symmetric Pairs

문제

You are given a table, Functions, containing two columns: X and Y.

Two pairs (X1, Y1) and (X2, Y2) are said to be symmetric pairs if X1 = Y2 and X2 = Y1.

Write a query to output all such symmetric pairs in ascending order by the value of X.

접근

문제는 1) x = y인 짝들과 2) x != y 인 짝들의 union으로 접근하였다.
우선, 1) x = y인 경우는 아래 쿼리로 표현할 수 있다.

주목해야하는 부분은 f1.x < f1.y인 부분으로, 뽑아내는 짝들의 x값이 y보다 작은 짝들만 찾아주게 되면 1) x = y인 부분은 제외할 수 있다. 이와 같은 두 쿼리를 union으로 묶어주고 마지막 order by를 통해 정렬만 하면 문제가 해결된다.
