Knapsack algorithm

  • 무게와 가치, 두 가지가 있고 정한 무게만큼의 최대 가치를 구할 때 사용하는 알고리즘
  • Greedy algorithm으로 풀 수 있지만 시간 복잡도가 2^N 이고 항상 최적의 해를 찾는다는 보장이 없다
  • 알고리즘
    • dp[i][j] : i번째 물건까지 탐색하였을 때, j무게를 가져갔을 때 획득의 가치의 최댓값
    • w[i], v[i] : i번째 물건의 무게/가치
      1. i번째 물건을 담은 경우
        • dp[i][j] = dp[i-1][j-w[i]] + v[i]
        • i-1번째 물건까지 탐색한 경우의 j무게에서 i번째 무게를 뺀 것
      2. i번째 물건을 담지 않은 경우
        • dp[i][j] = dp[i-1][j]
        • i번째 물건을 가져가지 않았기 때문에 무게는 그대로
