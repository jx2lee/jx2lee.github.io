[Hadoop] CDH(ClouDera Hadoop) Client 설정

Posted on 2020-05-15 In Hadoop Disqus:

Python 을 이용해 CDH 연동하는 과정에서 Hadoop Client 설정 방법에 대해 정리한다.

✨ Contents:

  1. Bianry Download
  2. CDH Hadoop Config
  3. Client 압축 해제
  4. Config 적용
  5. /etc/hosts 적용
  6. ENV

Bianry Download

  • wget 을 이용해 CDH hadoop 3 버젼을 다운
  • $ wget https://archive.apache.org/dist/hadoop/core/hadoop-3.0.0/hadoop-3.0.0.tar.gz

CDH Hadoop Config

  • CDH 웹에서 hadoop config 을 다운로드
    • core-site.xml
    • hdfs-site.xml
    • yarn-site.xml (Yarn 구성 시)

Client 압축 해제

  • 특정 directory 에 압축 해제
  • $ tar -xvf hadoop-3.0.0.tar.gz && mv hadoop-3.0.0 hadoop

Config 적용

  • $HADOOP_HOME/etc/hadoop 에 위에 준비한 CDH config 을 copy & paste
  • $ cp *-site.xml $HADOOP_HOME/etc/hadoop

/etc/hosts 적용

  • Client 서버의 IP hostname 에 마찬가지로 클러스터 노드 정보를 작성
    • example
    • $ touch /etc/hosts 로 파일 적용
