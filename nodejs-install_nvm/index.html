NVM?

  • node js 버전 매니저로 시스템에 여러 개의 nodejs 를 설치하고 사용할 버전을 쉽게 전환할 수록 도와주는 shell script
  • rvm(Ruby Version Manager) 와 비슷한 역할을 수행

Install NVM

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash

.bashrc (or .zsh)

Installing a specific version of Nodejs

Commands

  • search node version: nvm ls-remote
  • select specific node version: nvm use {node-version}
  • set default version: nvm alias default {node-version}
  • delete version: nvm uninstall {node-version}
